## Quick Start

### Try It Online
**[facegrid.juleslemee.com](https://facegrid.juleslemee.com)**

### Local Development
```bash
git clone https://github.com/juleslemee/FaceGrid.git
cd FaceGrid
npm install
npm run dev
```
